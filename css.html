Cascading Stylesheet

Introduction

CSS, otherwise known as Cascading StyleSheet, is a language used to apply styling to a website. This would help the website look more aesthetically pleasing. However, CSS can also be a little inefficient when coding. We cannot make variables to shorten our codes. Thus, we can use SCSS files. SCSS allows us to give variables to stylings. However, this cannot be directly processed by web browsers. So, we need to use SASS, a software that can compile SCSS files into CSS files. We can also update to GitHub and will directly be compiled into CSS.

CSS Selectors

CSS also has selectors. This is used when we want to specifically choose elements to be styled. Examples include:

CSS Selectors Examples Functions
.class1 .class2 .intro .par1 Selects all of par1 that are inside of intro
element.class p.intro Selects all <p> elements that has class name intro>
element, element div, p Selects all <div> and <p> elements
element element div p Selects all <p> that are inside of <div>
element + element div + p Selects all <p> that are placed after <div>
element > element div > p Selects all <p> that are direct descendants of <div>
