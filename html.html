What's HTML?

HTML, or Hypertext Markup Language, is a language that are used to make websites. The most current version is HTML5. HTML uses tags to tell the web browsers the properties of the text inside the tags.
The most basic layout of writing HTML for a website is like this:

Now, let's break down what all this means.

The <!DOCTYPE> tag is telling the browser that the HTML code used is HTML5

The <head> tag is used to tell the browser that this is the head of the webpage. Inside the <head> tag, you can put in your title, by using the <title> tag. You can also put styles, by typing in CSS codes inside a <style> tag, or linking the CSS stylesheet: <link rel="stylesheet" href="url">.

The <body> tag is where all the content of the webpage are. All text, images, videos, audios, are all put inside this tag.
